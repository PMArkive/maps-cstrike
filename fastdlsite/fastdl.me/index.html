fastdl.me

(WIP) big fat fastdl for Counter-Strike: Source maps

Directory list of canon/filtered maps | Directory list of all hashed/unfiltered maps

Smaller list of mostly bhop/trikz/kz/surf/xc maps

Contact:

  • Email: hello@fastdl.me

Everything from Gamebanana is included and it continues to be periodically checked (manually) for new maps.
You can find map info dumps used by this website at https://github.com/srcwr/maps-cstrike

How to request maps to be added/fixed:
  • Post in the Sourcejump Discord server or bhoptimer Discord server (main)
  • Send an email.

Questions:

  • Is it okay to scrape & download everything?

    • Yes. Cloudflare R2 bandwidth is free.
      Use `/maps/mapname.bsp.bz2` or `/hashed/123123123.bsp.bz2`.
      Also `python process.py` from maps-cstrike will make a sqlite database for querying mapname->hash [1].

  • Can I use this as my fastdl?

    • Yes, or you can make your fastdl server redirect maps to here.
      sv_downloadurl "http://main.fastdl.me/"

  • Maybe host .bsp files too?

    • Maybe one day but only .bsp.bz2's for now. It would be cool to mount the "maps" folder with rclone/sshfs/httpdirfs.

  • What's the difference between /maps/ & /hashed/?

    • /maps/: slightly curated list that uses the canon/correct maps and removes some maps.
      • Canon maps csv
      • Removed maps csvs
    • /hashed/: list of everything (old versions, corrupted files, etc)

  • Can I have subdomain.fastdl.me that does maps but redirects to my fastdl otherwise?

    • This is possible now. If you want one:
      Send me a preferred fastdl.me subdomain & a link to a URL to redirect non-maps to.

  • Can you host a CS:GO or TF2 fastdl?

    • I'm not personally interested in hosting maps for these right now as it'd cost more money.
      Here's a cool TF2 map archive though: https://maps.mevl2.duckdns.org/ (more games at https://ar.mevl2.duckdns.org/)
      And here's a cool, curated GMOD-oriented bhop fastdl: https://czarchasm.club/fastdl/
